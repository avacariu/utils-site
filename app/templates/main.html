Utilities

These are just some small utilities I made to help me out. They might be of some use to other people.

There is absolutely no formatting. Output is just sent to the browser with mimetype="text/plain".

  • IP address
  • User agent string
  • Request body
  • Request headers
  • md5sum. Pass text=TEXT as a parameter. Default is ''.
  • SHA sums. Pass ver=256&text=TEXT as parameters. Available versions are 1, 224, 256, 384 and 512. Default is sha1('').
