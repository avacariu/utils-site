Utilities

These are just some small utilities I made to help me out. They might be of some use to other people.

There is absolutely no formatting. Output is just sent to the browser with mimetype="text/plain".

Utilities hosted on this site

IP address (GET)
Responds your external IP address.
User agent string (GET)
Responds with your user agent string
Request body (GET | POST)
Responds with the body of the POST request you just sent.
Request headers (GET | POST)
Responds with the headers of the request you just sent.
md5sum (GET | POST)
Computes MD5 hash. Pass text=TEXT as a parameter. Default is ''.
SHA sums (GET | POST)
Computes SHA sums. Pass ver=256&text=TEXT as parameters. Available versions are 1, 224, 256, 384 and 512. Default is sha1('').
Fortune (GET)
Responds with whatever fortune outputs.
