Utilities

These are a few miscellaneous things I made to help me out; some just for fun.

Useful stuff

There is absolutely no formatting here; output is sent to the browser with mimetype="text/plain".

User agent string (GET)
Responds with your user agent string
Request body (GET | POST)
Responds with the body of the POST request you just sent.
Request headers (GET | POST)
Responds with the headers of the request you just sent. Note that since this is behind CloudFlare and hosted on Heroku, there might be more headers than you expect.
Online check (GET | POST)
Check if you have internet access by sending a request to /amionline/RANDOMSTRING and checking if the response is exactly RANDOMSTRING.
Pomodoro-style timer
Keeps you on track.
